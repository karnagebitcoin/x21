# x21-dan vs x21.social - Settings Comparison Summary

## What We Have (x21-dan)

### ✅ Comprehensive Settings Categories
1. **General** - Interface preferences, language, modes
2. **Content & Privacy** - Web of Trust, spam filters, media policies, muting
3. **Appearance** - Themes, typography, layouts, styling (VERY extensive)
4. **Widgets** - Widget management
5. **Relays** - Relay configuration
6. **Backup & Sync** - NIP-78 sync
7. **AI Tools** - AI features
8. **Translation** - Translation services
9. **Wallet** - Lightning/Zap settings
10. **Post Settings** - Posting preferences

### 🎨 Notable Strengths
- **Advanced Theming**: 5 theme modes, 6 color palettes, 8+ primary colors
- **Typography Control**: Font family, size sliders for text/titles/logo
- **Layout Flexibility**: 3 layout modes (Boxed, Full Width, Island) + Deck view
- **Granular Styling**: Separate radius controls for buttons, cards, and media
- **Web of Trust**: 4-level system with notification filtering
- **Spam Protection**: Hashtag and mention spam filters
- **Media Policies**: 5 different auto-load policies
- **Custom Logo**: Text or image with size controls
