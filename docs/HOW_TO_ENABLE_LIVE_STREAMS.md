# How to Enable Live Streams

The Live Streams feature is disabled by default. Here's how to enable it:

## Steps to Enable

1. **Open Settings**
   - Click on your profile avatar in the sidebar
   - Select "Settings" from the menu
   - OR navigate directly to Settings from the sidebar

2. **Go to Appearance Settings**
   - In the Settings page, click on "Appearance"

3. **Enable Live Streams in Navigation**
   - Scroll to the "Navigation Menu Items" section
   - Find "Live Streams" in the list
   - Toggle the switch to **ON** (enabled)

4. **Reorder (Optional)**
   - You can drag the "Live Streams" item to reorder it
   - Position it wherever you prefer in your navigation menu

5. **Return to Home**
   - The "Live Streams" option will now appear in your left sidebar
   - Click on it to view active live events

## Usage

### Viewing Live Streams
- Click on "Live Streams" in the sidebar
- Browse active live events
- Events are sorted by number of viewers
- Click on any event to join

### Joining a Live Stream
- Click on a live event card
- View the stream (if video URL is provided)
- See current viewers count
- View recent zaps to the stream

### Participating in Live Chat
- Scroll to the "Live Chat" section
- Type your message in the text box
- Press Enter or click Send
- Your message appears in real-time

### Zapping
- **Zap the Stream**: Click "Zap Stream" button
- **Zap a Message**: Hover over a chat message and click the zap icon
- Zaps appear in the horizontal scrollable list

## What You'll See

### Live Streams Page
- List of currently live events
- Red "LIVE" badge on active streams
- Thumbnails, titles, and descriptions
- Host information
- Current viewer count
- Start time

### Live Stream Detail Page
- Stream video player (if available)
- Host information
- Event description
- Viewer count
- Recent zaps (horizontal scroll)
- Live chat with real-time messages
- Message input to participate

## Requirements

- **Login Required**: You must be logged in to send chat messages
- **Relays**: The app subscribes to major relays to find live events
- **NIP-53 Support**: Events must follow NIP-53 (Live Activities) standard

## Troubleshooting

**Not seeing any live streams?**
- There may not be any active streams at the moment
- Try refreshing the page
- Check back later

**Can't send messages?**
- Make sure you're logged in
- Check your relay connections in Settings > Relays

**Video not playing?**
- Some streams may only provide chat (no video URL)
- Try a different browser if video doesn't work
- Check that the stream URL is valid

## Privacy & Safety

- All chat messages are public and stored on Nostr relays
- Anyone can view your messages
- Be respectful and follow community guidelines
- You can mute users from their profile if needed
